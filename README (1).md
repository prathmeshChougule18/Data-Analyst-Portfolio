## ✨ Features

- 📊 **Interactive Charts** — Bar, Radar, Donut, Line, Sparkline charts via Chart.js
- 🧭 **Sidebar Navigation** — Fixed sidebar like a real BI dashboard (Tableau/Power BI feel)
- 🟢 **Live Clock** — Real-time clock in the topbar
- 📈 **Animated Skill Bars** — Scroll-triggered animated progress bars
- 🃏 **KPI Metric Cards** — Highlighting key career stats
- 🚀 **Project Showcase** — 3 live projects with GitHub + Live Demo links
- 💼 **Experience Timeline** — With mini sparkline charts per role
- 📱 **Responsive** — Mobile-friendly layout
- 🎨 **Dark BI Theme** — Custom dark palette with teal accent

## 🚀 How to Run Locally

```bash
# Clone the repo
git clone https://github.com/prathmeshChougule18/portfolio.git

# Open in browser
cd portfolio
open index.html
```

No server needed — just open `index.html` in any browser.

---

## 🌐 Deploy on GitHub Pages

1. Go to your repo → **Settings** → **Pages**
2. Under **Source**, select `main` branch → `/ (root)` → **Save**
3. Your site will be live at:
   ```
   https://prathmeshChougule18.github.io/portfolio
   ```
