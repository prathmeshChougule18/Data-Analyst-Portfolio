## 🚀 Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/prathmeshChougule18/portfolio.git

# 2. Open in browser
cd portfolio
open index.html      # macOS
start index.html     # Windows
```

No install needed. Works in any modern browser.

---

## 🌐 Deploy on GitHub Pages (Free Hosting)

```
1. Push index.html + README.md to your repo (main branch)
2. Go to: Repo → Settings → Pages
3. Source → Deploy from branch → main → / (root)
4. Click Save
5. Live in ~2 minutes ✅
```
